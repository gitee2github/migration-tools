# migration-tools

#### 介绍
迁移工具是由统信公司自主研发，具有CentOS7、8系列系统迁移至OpenEuler系统功能的迁移工具。

#### 依赖

* python3
* /bin/bash


#### 安装教程

本工具主要由python和shell构成，开箱即用。

#### 使用说明

1.  mkdir /mnt/iso
2.  将1002a/1050a镜像文件挂载至/mnt/iso目录下
3.  执行 `python3 xxx.py`
